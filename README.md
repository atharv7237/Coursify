# 🚀 Coursify – AI-Powered Learning Roadmap Generator

Coursify is a full-stack web application that helps users generate personalized learning roadmaps using Artificial Intelligence.

Users can create an account, log in securely, enter a learning goal (such as becoming a MERN Developer, Data Analyst, AI Engineer, etc.), and instantly receive a structured roadmap generated by Google's Gemini AI.

The generated roadmaps are stored in MongoDB and can be accessed anytime from the user's dashboard.

* Logout Functionality

### 🤖 AI Roadmap Generation

* Dynamic Prompt Generation
* Google Gemini API Integration
* Structured JSON Response Generation
* Personalized Learning Plans
* Goal-Based Roadmap Creation

### 📚 Roadmap Management

* Save Generated Roadmaps
* View All Generated Roadmaps
* Individual Roadmap Pages
* Structured Learning Milestones
* Resources and Project Recommendations

### 📊 Roadmap Structure

Each roadmap contains:

* Roadmap Overview
* Skills Gained
* Learning Milestones
* Topics to Learn
* Why Each Topic Matters
* Recommended Resources
* Mini Projects
* Final Capstone Project
* Expected Outcome

## ⚙️ Installation

### Clone Repository

```bash
git clone <repository-url>
cd coursify
```

### Install Dependencies

```bash
npm install
```

### Create Environment Variables

Create a `.env` file in the root directory:

```env
MONGO_URI=your_mongodb_connection_string
GEMINI_API_KEY=your_gemini_api_key
JWT=your_secret_key
```

### Run Application

```bash
npm start
```

or

```bash
nodemon app.js
```
